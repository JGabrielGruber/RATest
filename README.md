# RATest
REST API Test

## Implementation

### Dependencies

* Python 3.7
* Python-pip

### Installation
On your terminal:

- Clone the repository:
```
$ git clone https://github.com/JGabrielGruber/RATest.git
```
- Go to the project folder:
```
$ cd RATest
```
- Change your source:
```
$ source bin/activate
```
- Inside the repository, run:
```
$ pip install -r package.lock 
```

### Usage

To start the server, run:
```
$ python manage.py runserver
```
